== Description ==

"SX AS3 Media" is FREE media storage plugin.
Allows easily connect Amazon S3 bucket to any wordpress site in minutes and use it as alternative/backup media storage.

== Installation ==
1. Install the plugin through the WordPress plugins screen directly or Upload the plugin files to the `/wp-content/plugins/plugin-name` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress
